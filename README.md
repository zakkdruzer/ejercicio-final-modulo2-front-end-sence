# Weather Chile - Módulo 2 App de Clima

Proyecto frontend desarrollado como parte del **Módulo 2** de la app de clima.

## Descripción

Weather Chile es una aplicación frontend construida con **HTML5 semántico**, **Bootstrap 5**, **CSS3** y **JavaScript**.

La app permite:

- visualizar una grilla de localidades en la vista principal,
- consultar el clima actual de cada ciudad,
- mostrar el detalle de la localidad seleccionada,
- revisar temperatura, humedad, viento y pronóstico semanal.

## Funcionalidades implementadas

- Home con 10 localidades.
- Cards Bootstrap con:
  - nombre de la ciudad,
  - región,
  - icono del clima,
  - temperatura,
  - estado actual.
- Vista detalle dentro de la misma página.
- Navegación controlada con JavaScript.
- Pronóstico semanal por ciudad.
- Diseño responsive para móvil y escritorio.
- Estructura semántica con:
  - `header`
  - `nav`
  - `main`
  - `section`
  - `article`
  - `footer`

## Tecnologías utilizadas

- HTML5
- CSS3
- Bootstrap 5
- JavaScript

## Cómo ejecutar el proyecto

1. Descarga o clona este repositorio.
2. Abre el archivo `index.html` en tu navegador.
3. Haz clic en una ciudad para ver su detalle y pronóstico semanal.
