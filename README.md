# Weather Chile - Módulo 2 App de Clima

Proyecto frontend desarrollado como parte del Módulo 2.

## Descripción

Este proyecto corresponde a una aplicación de clima en su primera versión (MVP), construida con HTML5 semántico, Bootstrap y JavaScript.

## Tecnologías utilizadas

- HTML5
- CSS3
- Bootstrap 5
- JavaScript

## Estructura inicial del proyecto

```bash
weather-frontend-m2/
├── index.html
├── README.md
├── css/
│   └── styles.css
└── js/
    └── main.js
```

## Cómo ejecutar

1. Descarga o clona este repositorio.
2. Abre el archivo `index.html` en tu navegador.
